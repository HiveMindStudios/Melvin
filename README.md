## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- A computer
- Internet connection
- Python
- Discord Account

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/VectorKappa/Melvin.git
   ```

2. Install the python modules from PyPi

   ```sh
   pip install -r requirements.txt
   ```
